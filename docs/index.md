# goldfishmem

A production grade memory system for agents.

The Python import name matches the distribution name:
> `from goldfishmem import ...`.

## Install

```bash
pip install goldfishmem
# or with uv
uv add goldfishmem
```

## Quick start

```python
from goldfishmem import hello_world

print(hello_world())            # "Hello, world!"
print(hello_world("goldfish"))  # "Hello, goldfish!"
```
